java 中新建的对象在”堆”中.

接口相关

  • 接口可以声明引用(line 6)，但不能new来实例化(line 7):
  • 接口引用可以指向实现类的实例:

但要清楚的是接口引用调用其实现类的方法时,该方法必须已经在接口中被声明，而且在接口的实现类中该实现方法的类型和参数必须与接口中所定义的精确匹配

  • 接口的域:都是静态常量，public static final

    声明的同时就必须赋值（接口中不能编写静态代码块）

  • 接口的方法:都是抽象方法，public abstract（缺省情况下也是），不可含有static或final

  • 接口没有构造器，与之相对的抽象类可以有。
  • 同时，接口不能有静态方法，抽象类可以有
  • 接口不能实现另一个接口，但可以继承，且可以继承多个接口
